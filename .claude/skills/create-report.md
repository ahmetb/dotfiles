## Style 

Print

The design works on paper. @media print strips the grain overlay and forces the background to white. All other elements (rules, hairlines, ink colour, generous margins)
translate directly to a printed page. If a document doesn't print well, it isn't really a dossier.

---
Skill triggers

When invoked with prompts like:
- "Write an incident report / RCA / post-mortem"
- "Produce a field note / dossier / engineering brief"
- "Document an investigation"
- "Write up a technical analysis"
- "Make this look like a serious document, not a dashboard"

…apply this system in full: Fraunces + Newsreader + JetBrains Mono, paper-warm palette, single signal colour, masthead + numbered sections + colophon, hero SVG diagram,
fact grid of three giant numerals, pull quote (used at most twice), citation table, timeline, mitigations grid.
